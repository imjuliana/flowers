# Julia's Craft - Handmade Flowers E-Commerce

Welcome to Julia's Craft, your online shop for beautiful, handcrafted floral arrangements.

## Features

- Browse a collection of 9 unique flower bouquets
- Add products to cart and checkout
- Customer registration, login, profile, and order history
- Seller dashboard for managing orders, products, and customers
- Order tracking
- Responsive design

## Getting Started

1. **Clone the repository:**
   ```sh
   git clone https://github.com/imjuliana/flowers-ecommerce.git
   ```
2. **Open the project folder in your editor (e.g., VS Code).**
3. **Run locally:**
   - Open `index.html` in your browser.

## Folder Structure

```
flowers-ecommerce/
│
├── index.html
├── login.html
├── profile.html
├── seller-dashboard.html
├── shop.html
├── track-order.html
├── flower/
│   └── [images: .jpg, .png, etc.]
```
- `index.html` - Main landing page
- `login.html` - Customer and seller login/registration
- `profile.html` - Customer dashboard/profile
- `seller-dashboard.html` - Seller dashboard
- `shop.html` - Product listing/shop
- `track-order.html` - Order tracking
- `flower/` - Images for products, logo, owner, etc.
